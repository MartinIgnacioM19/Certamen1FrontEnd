# Portafolio Profesional - Martín Mondaca

Este proyecto es una página web tipo portafolio personal desarrollada para la asignatura de Desarrollo Web Front-End.

## Descripción
Sitio web de una sola página (Single Page) que presenta mi perfil profesional, habilidades técnicas y proyectos destacados como estudiante de Analista Programador y Ciberseguridad.

## Tecnologías Usadas
* [cite_start]**HTML5**: Uso de etiquetas semánticas, tablas y formularios[cite: 13].
* [cite_start]**CSS3**: Diseño responsivo con Flexbox, Grid y Media Queries[cite: 22].
* [cite_start]**GitHub**: Control de versiones y despliegue del proyecto[cite: 30].

## Diseño Responsivo
El portafolio cuenta con un diseño adaptativo para:
* **Desktop**: Diseño completo con múltiples columnas.
* **Tablet**: Ajustes para pantallas de hasta 768px.
* **Móvil**: Adaptación completa para pantallas de hasta 480px.

## 💻 Cómo ejecutar el proyecto
1. Clonar el repositorio.
2. Abrir el archivo `index.html` en Visual Studio Code.
3. Se recomienda el uso de la extensión Live Preview para visualizar los cambios en tiempo real directamente en el editor o en el navegador.
